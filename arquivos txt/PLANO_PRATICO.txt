================================================================================
                      SEMANA 1: SETUP FIREBASE
                      (5-7 Dias de Trabalho)
================================================================================

OBJETIVO: Ter projeto Firebase criado, autenticação funcionando

□ DIA 1-2: CRIAR PROJETO FIREBASE

  TAREFAS:
  1. Ir para https://console.firebase.google.com
  2. Clique em "Criar Projeto"
  3. Nome: "addevos-saas"
  4. Google Analytics: Ativar (recomendado)
  5. Aguardar 2-3 minutos

  RESULTADO ESPERADO:
  - Ter Project ID: addevos-saas
  - Ter Firebase Credentials (guardar em local seguro)

  TEMPO ESTIMADO: 30 minutos

---

□ DIA 2-3: ATIVAR SERVIÇOS FIREBASE

  TAREFAS:
  
  A) AUTENTICAÇÃO:
     1. Firebase Console > Authentication
     2. "Get Started"
     3. Ativar "Email/Password"
     4. Ativar "Google"
     5. Salvar

  B) FIRESTORE DATABASE:
     1. Firebase Console > Firestore Database
     2. "Create Database"
     3. Modo: "Start in test mode" (depois mudar regras)
     4. Localização: us-central1
     5. Criar

  C) CLOUD STORAGE:
     1. Firebase Console > Storage
     2. "Get Started"
     3. Localização: us-central1
     4. Criar

  D) OBTER CREDENCIAIS:
     1. Project Settings (engrenagem)
     2. "Suas Apps"
     3. Registrar novo app Web
     4. Copiar credenciais Firebase Config
     5. Guardar em arquivo seguro

  RESULTADO ESPERADO:
  - Ter credenciais Firebase copiadas
  - Todos os serviços ativos

  TEMPO ESTIMADO: 1 hora

---

□ DIA 4-5: INSTALAR DEPENDÊNCIAS

  TAREFAS:
  1. Abrir Terminal na pasta do projeto
  
  2. Instalar Vue 3:
     npm init vue@latest addvos-web
     
  3. Instalar Firebase:
     npm install firebase pinia vue-router
     
  4. Dependências extras:
     npm install stripe axios date-fns

  RESULTADO ESPERADO:
  - Pasta "addvos-web" criada
  - node_modules com todas dependências
  - Package.json atualizado

  TEMPO ESTIMADO: 30 minutos (download)

---

□ DIA 6-7: CONFIGURAR FIREBASE SDK

  TAREFAS:
  
  1. Criar arquivo: src/config/firebase.js
  
  2. Copiar código (veja IMPLEMENTACAO_CODIGO.md - seção "Setup Firebase")
  
  3. Preencher credenciais:
     - apiKey
     - projectId
     - storageBucket
     - Etc...
  
  4. Testar conexão:
     npm run dev
     Abrir http://localhost:5173

  RESULTADO ESPERADO:
  - App Vue rodando sem erros
  - Conexão com Firebase funcionando
  - Console sem erros de conexão

  TEMPO ESTIMADO: 1 hora

RESUMO SEMANA 1:
  ✓ Projeto Firebase criado
  ✓ Serviços ativos (Auth, Firestore, Storage)
  ✓ SDK instalado e configurado
  ✓ App Vue rodando
